# Microsoft Edge policy mapping overview

To help you migrate your policies from Microsoft Edge Legacy (version 45 and earlier) or from Google Chrome (version 79) to the new version of Microsoft Edge we are providing policy mapping tables. These tables are in the following articles:

- [Microsoft Edge Legacy to Microsoft Edge policy mapping](microsoft-edge-policy-map-legacy-to-newedge.md)
- [Google Chrome to Microsoft Edge policy mapping](microsoft-edge-policy-map-chrome-to-newedge.md)

> [!NOTE]
> The mapping documents are meant to help with your initial deployment of Microsoft Edge and only the Microsoft Edge 80 and earlier policies are mapped. For a definitive list of all the latest policies, see:
> - [Browser policy reference](microsoft-edge-policies.md)
> - [Update policy reference](microsoft-edge-update-policies.md)
